# Amp Code

## Overview

Amp Code uses the same commands format as Claude Code. The installer deploys workflows and commands to `.amp/commands/`.

## Installation

Run the installer with the `amp` tool target:

```bash
TOOLS=amp ./install.sh
```

This copies all available commands and workflows into `.amp/commands/` in your project.

## Usage

Once installed, use commands and workflows the same way you would in Claude Code. Amp Code reads from `.amp/commands/` and makes them available in your sessions.
