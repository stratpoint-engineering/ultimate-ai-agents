# Quick Start

## One-Liner Install

```bash
curl -fsSL https://raw.githubusercontent.com/stratpoint/ultimate-ai-agents/main/install.sh | bash
```

This auto-detects which tools you have installed and deploys agents to each one.

## From Cloned Repo

```bash
git clone https://github.com/stratpoint/ultimate-ai-agents.git
cd ultimate-ai-agents
./install.sh
```

When run without env vars in a terminal, the installer walks you through every option interactively.

## Non-Interactive

Set env vars to skip prompts:

```bash
TOOLS=claude DEPT=engineering ./install.sh
```

## What Gets Installed

| Tool | What's deployed | Location |
|------|----------------|----------|
| Claude Code | Agents, workflows, rules, skills | `.claude/` |
| Cursor | Rules (.mdc) | `.cursor/rules/` |
| Codex CLI | Rules as AGENTS.md | `./AGENTS.md` or `~/.codex/` |
| Amp Code | Workflow commands | `.amp/commands/` |
| Windsurf | Rules appended to global_rules.md | `.codeium/windsurf/` |
| Gemini CLI | Agent commands (TOML) | `.gemini/commands/` |
