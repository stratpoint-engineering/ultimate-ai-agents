# Uninstall

## Interactive

```bash
./uninstall.sh
```

Prompts for scope and which tools to clean.

## Non-Interactive

```bash
# Remove from current project
TOOLS=all SCOPE=project ./uninstall.sh

# Remove global install
SCOPE=global ./uninstall.sh

# Remove only Claude Code agents
TOOLS=claude ./uninstall.sh

# Remove Claude + Cursor
TOOLS=claude,cursor ./uninstall.sh
```

## Remote (Without Cloning)

```bash
curl -fsSL https://raw.githubusercontent.com/stratpoint/ultimate-ai-agents/main/uninstall.sh | bash
```

## What Gets Removed

| Tool | What's removed | What's kept |
|------|---------------|-------------|
| Claude Code | `agents/`, `commands/`, `rules/`, `skills/` | `settings.json` (may have custom config) |
| Cursor | `.cursor/rules/` | |
| Codex CLI | `AGENTS.md` (only if ours) | User-authored AGENTS.md |
| Amp Code | `.amp/commands/` | |
| Windsurf | Our section in `global_rules.md` | Rest of the file |
| Gemini CLI | `.gemini/commands/` | |
