# Ultimate AI Agents

An organization-wide library of **AI agent definitions**, **engineering workflows**, and shared rules — deployable to every major AI coding tool in one command.

## Supported Platforms

| Type | Platforms |
|------|-----------|
| **CLI Tools** | Claude Code, Cursor, Codex CLI, Amp Code, Windsurf, Gemini CLI |
| **Browser/Desktop** | Claude.ai, ChatGPT, Gemini, Google AI Studio, any LLM |

## Departments Covered

| Department | Focus |
|------------|-------|
| Engineering | Architecture, cross-cutting, JavaScript, savants |
| Product & Design | UI/UX, research, writing |
| Project Management | PM, Scrum, BA |
| Sales | Strategy, pre-sales, accounts |
| Marketing | Content, digital, brand, social |
| HR | Generalist, recruiting, people ops |
| L&D | Training, knowledge, coaching |
| Finance | Analysis, procurement |
| Operations | Ops management, QA |
| Admin | Executive support, comms |
| Leadership | Strategy, chief of staff |

## Quick Start

```bash
# Clone the repo
git clone https://github.com/stratpoint/ultimate-ai-agents.git
cd ultimate-ai-agents

# Interactive install
./install.sh

# Or non-interactive
TOOLS=claude DEPT=engineering ./install.sh
```

Or use the one-liner:

```bash
curl -fsSL https://raw.githubusercontent.com/stratpoint/ultimate-ai-agents/main/install.sh | bash
```
