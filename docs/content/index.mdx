## Quick Start

### CLI Tools

```bash
curl -fsSL https://raw.githubusercontent.com/stratpoint-engineering/ultimate-ai-agents/main/install.sh | bash
```

Or clone and run interactively:

```bash
git clone https://github.com/stratpoint-engineering/ultimate-ai-agents.git
cd ultimate-ai-agents
./install.sh
```
