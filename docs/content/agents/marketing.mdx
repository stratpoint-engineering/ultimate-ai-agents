# Marketing

Agents for marketing teams covering content strategy, digital campaigns, brand management, and social media.

## Agent Directory

| Agent | Purpose | Example Use |
|-------|---------|-------------|
| `content-strategist` | Plan content calendars, develop messaging frameworks, audit content | "Create a Q2 content calendar aligned with our product launches" |
| `digital-marketer` | Plan and optimize digital campaigns, analyze performance metrics | "Design an A/B test plan for our landing page conversion funnel" |
| `brand-manager` | Maintain brand consistency, develop guidelines, manage brand assets | "Audit our website copy for brand voice consistency" |
| `social-media-manager` | Plan social content, manage community engagement, track social metrics | "Draft a week of LinkedIn posts for our developer tools launch" |

## How to Use

### Claude Code

```bash
@content-strategist Develop a thought leadership series on AI adoption
@digital-marketer Analyze these campaign metrics and recommend optimizations
@brand-manager Review this landing page copy for brand alignment
@social-media-manager Create a social launch plan for the new feature
```

### ChatGPT

Start a conversation with the agent's system prompt loaded as a Custom GPT:

```
You are a content-strategist agent. [system prompt]
Develop a thought leadership series on AI adoption.
```

### Other Platforms

- **Cursor**: Available as rule files in `.cursor/rules/`
- **Gemini**: Available as gems in `agents/gemini/`
- **Codex**: Available as agents in `agents/codex/`
