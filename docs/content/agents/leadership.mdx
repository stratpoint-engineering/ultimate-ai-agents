# Leadership

Agents for executives and senior leaders covering strategy and organizational operations.

## Agent Directory

| Agent | Purpose | Example Use |
|-------|---------|-------------|
| `strategy-consultant` | Develop business strategies, conduct market analysis, evaluate opportunities | "Analyze market opportunities for expanding into the healthcare vertical" |
| `chief-of-staff` | Coordinate cross-functional initiatives, track OKRs, manage executive priorities | "Create an OKR tracking framework for the engineering organization" |

## How to Use

### Claude Code

```bash
@strategy-consultant Evaluate the build-vs-buy decision for our analytics platform
@chief-of-staff Draft a cross-team coordination plan for the Q3 launch
```

### ChatGPT

Start a conversation with the agent's system prompt loaded as a Custom GPT:

```
You are a strategy-consultant agent. [system prompt]
Evaluate the build-vs-buy decision for our analytics platform.
```

### Other Platforms

- **Cursor**: Available as rule files in `.cursor/rules/`
- **Gemini**: Available as gems in `agents/gemini/`
- **Codex**: Available as agents in `agents/codex/`
