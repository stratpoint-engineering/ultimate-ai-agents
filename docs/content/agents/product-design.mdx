# Product Design

Agents for product designers, UX professionals, and design system teams. They help with user research, interaction design, and content design.

## Agent Directory

| Agent | Purpose | Example Use |
|-------|---------|-------------|
| `product-designer` | Design user interfaces, flows, and design systems | "Design a checkout flow that reduces cart abandonment" |
| `ux-researcher` | Plan and analyze user research studies | "Create a usability test plan for the new onboarding experience" |
| `ux-writer` | Write microcopy, UI text, and content guidelines | "Write error messages and empty states for the dashboard" |

## How to Use

### Claude Code

```bash
@product-designer Design a settings page with accessibility in mind
@ux-researcher Create an interview script for testing our search feature
@ux-writer Write onboarding tooltip copy for new users
```

### ChatGPT

Start a conversation with the agent's system prompt loaded as a Custom GPT, or paste it at the start:

```
You are a product-designer agent. [system prompt]
Design a settings page with accessibility in mind.
```

### Other Platforms

- **Cursor**: Available as rule files in `.cursor/rules/`
- **Gemini**: Available as gems in `agents/gemini/`
- **Codex**: Available as agents in `agents/codex/`
