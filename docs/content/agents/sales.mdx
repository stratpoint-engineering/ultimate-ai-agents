# Sales

Agents for sales teams, pre-sales engineers, and account managers. They help with strategy, technical proposals, requirements documentation, and client relationship management.

## Agent Directory

| Agent | Purpose | Example Use |
|-------|---------|-------------|
| `sales-strategist` | Develop sales strategies, qualify leads, plan deal cycles | "Analyze this prospect's tech stack and recommend our positioning" |
| `pre-sales-engineer` | Create technical proposals, demos, and proof-of-concept plans | "Draft a technical proposal for the enterprise SSO integration" |
| `brd-generator` | Generate source-verified BRDs from RFPs, RFIs, TORs, Solutions Documents, and Estimation Sheets | "Create a BRD from the RFP and solution doc in docs/input/" |
| `account-manager` | Manage client relationships, renewals, and expansion opportunities | "Prepare a quarterly business review deck for our top account" |

## How to Use

### Claude Code

```bash
@sales-strategist Build a competitive analysis for the fintech vertical
@pre-sales-engineer Create a technical architecture for the client's RFP
@brd-generator Create a BRD from the RFP and solution doc in docs/input/
@account-manager Draft a renewal proposal with upsell recommendations
```

### ChatGPT

Start a conversation with the agent's system prompt loaded as a Custom GPT:

```
You are a sales-strategist agent. [system prompt]
Build a competitive analysis for the fintech vertical.
```

### Other Platforms

- **Cursor**: Available as rule files in `.cursor/rules/`
- **Gemini**: Available as gems in `agents/gemini/`
- **Codex**: Available as agents in `agents/codex/`
