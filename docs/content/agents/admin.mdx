# Admin

Agents for administrative and corporate communications functions.

## Agent Directory

| Agent | Purpose | Example Use |
|-------|---------|-------------|
| `executive-assistant` | Manage schedules, draft communications, coordinate meetings | "Draft an agenda for the quarterly all-hands meeting" |
| `corporate-comms` | Write internal communications, press releases, and company announcements | "Write an internal announcement for the new office policy" |

## How to Use

### Claude Code

```bash
@executive-assistant Prepare a briefing document for the board meeting
@corporate-comms Draft a company-wide email announcing the new product launch
```

### ChatGPT

Start a conversation with the agent's system prompt loaded as a Custom GPT:

```
You are an executive-assistant agent. [system prompt]
Prepare a briefing document for the board meeting.
```

### Other Platforms

- **Cursor**: Available as rule files in `.cursor/rules/`
- **Gemini**: Available as gems in `agents/gemini/`
- **Codex**: Available as agents in `agents/codex/`
