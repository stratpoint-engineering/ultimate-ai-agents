# Operations

Agents for operations teams covering process management and quality assurance.

## Agent Directory

| Agent | Purpose | Example Use |
|-------|---------|-------------|
| `operations-manager` | Optimize workflows, manage resources, improve operational efficiency | "Map our release process and identify bottlenecks" |
| `quality-assurance` | Define quality standards, audit processes, drive continuous improvement | "Create a quality checklist for our client deliverables" |

## How to Use

### Claude Code

```bash
@operations-manager Analyze our incident response process for improvements
@quality-assurance Design a QA framework for our consulting engagements
```

### ChatGPT

Start a conversation with the agent's system prompt loaded as a Custom GPT:

```
You are an operations-manager agent. [system prompt]
Analyze our incident response process for improvements.
```

### Other Platforms

- **Cursor**: Available as rule files in `.cursor/rules/`
- **Gemini**: Available as gems in `agents/gemini/`
- **Codex**: Available as agents in `agents/codex/`
