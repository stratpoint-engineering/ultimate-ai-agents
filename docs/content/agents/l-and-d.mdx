# Learning & Development

Agents for L&D professionals covering training design, knowledge management, and coaching.

## Agent Directory

| Agent | Purpose | Example Use |
|-------|---------|-------------|
| `training-designer` | Design training programs, workshops, and learning paths | "Create a 4-week training program for new engineering hires" |
| `knowledge-manager` | Organize knowledge bases, improve documentation systems | "Design a knowledge base taxonomy for our internal wiki" |
| `coach` | Provide professional development guidance and feedback frameworks | "Create a coaching plan for a mid-level engineer moving to tech lead" |

## How to Use

### Claude Code

```bash
@training-designer Build a workshop curriculum on API design best practices
@knowledge-manager Audit our docs and recommend a new information architecture
@coach Draft development goals for a product manager promotion track
```

### ChatGPT

Start a conversation with the agent's system prompt loaded as a Custom GPT:

```
You are a training-designer agent. [system prompt]
Build a workshop curriculum on API design best practices.
```

### Other Platforms

- **Cursor**: Available as rule files in `.cursor/rules/`
- **Gemini**: Available as gems in `agents/gemini/`
- **Codex**: Available as agents in `agents/codex/`
